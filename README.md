# NEXUS AI Chat

## Files
- `ai-setup.html` - Setup page to enter API key
- `nexus-chat.html` - Main AI chat interface

## Usage
1. Open `ai-setup.html` in browser
2. Enter your OpenRouter API key
3. Start chatting with Claude AI!

## API Key
Get free API key at: https://openrouter.ai/keys

## Deploy to Netlify
1. Connect GitHub repo to Netlify
2. Deploy master branch
3. Done!
